## Tutorial 1: Getting Started

**Duration**: 5 minutes  
**Target Audience**: New users, healthcare operations staff  
**Prerequisites**: None

### Script

**[INTRO - 0:00-0:30]**

*[Screen: RosterIQ logo and login page]*

**Narrator**: "Welcome to RosterIQ AI Agent - your intelligent assistant for healthcare provider roster operations. In this tutorial, you'll learn how to log in, navigate the interface, and ask your first question."

**[SECTION 1: Logging In - 0:30-1:00]**

*[Screen: Login page with credentials being entered]*

**Narrator**: "Start by navigating to your organization's RosterIQ URL and entering your credentials. Your IT administrator will have provided these to you."

*[Action: Click login button]*

**Narrator**: "Once logged in, you'll see the three-panel interface that makes RosterIQ easy to use."

**[SECTION 2: Understanding the Interface - 1:00-2:30]**

*[Screen: Three-panel layout with highlights]*

**Narrator**: "Let's explore the interface. On the left, you have the Query Input panel where you'll type your questions in plain English."

*[Highlight: Left panel]*

**Narrator**: "The center panel displays your results - including AI-generated insights, charts, and visualizations."

*[Highlight: Center panel]*

**Narrator**: "And on the right, you'll find detailed information including data sources, evidence, and recommendations."

*[Highlight: Right panel]*

**Narrator**: "Notice the session information at the top. RosterIQ remembers your work and will show you what changed since your last visit."

*[Highlight: Session info banner]*

**[SECTION 3: Your First Query - 2:30-4:30]**

*[Screen: Query input box with cursor]*

**Narrator**: "Let's ask your first question. Simply type in natural language - no SQL or technical commands needed."

*[Action: Type "What are the main issues with our roster processing?"]*

**Narrator**: "As you type, RosterIQ may suggest completions or related questions. Press Enter or click Submit to run your query."

*[Action: Submit query]*

*[Screen: Real-time streaming showing analysis steps]*

**Narrator**: "Watch as RosterIQ analyzes your question in real-time. You'll see each step of the analysis process, including what data sources are being queried."

*[Screen: Results appear with confidence score]*

**Narrator**: "Within seconds, you'll receive a comprehensive answer with a confidence score. The higher the confidence, the more certain RosterIQ is about its findings."

*[Highlight: Confidence score of 0.85]*

**Narrator**: "Notice the sources panel on the right - every piece of data is fully attributed so you can verify the analysis."

**[SECTION 4: Next Steps - 4:30-5:00]**

*[Screen: Query history and suggested queries]*

**Narrator**: "Your query is saved in the history panel, so you can easily return to it later. RosterIQ also suggests related questions you might want to explore."

**Narrator**: "In the next tutorial, we'll learn how to run diagnostic procedures for deeper analysis. Thanks for watching!"

*[Screen: RosterIQ logo with "Next: Diagnostic Procedures"]*

### Production Notes

- **Screen Recording**: Capture at 1920x1080, 30fps
- **Voiceover**: Professional narrator, clear and friendly tone
- **Music**: Subtle background music, healthcare/technology theme
- **Captions**: Include closed captions for accessibility
- **Branding**: RosterIQ logo in corner, consistent color scheme
- **Pacing**: Allow 2-3 seconds after each major point for comprehension

### Key Takeaways for Viewers

1. RosterIQ uses a three-panel interface for easy navigation
2. Ask questions in natural language - no technical skills needed
3. Real-time streaming shows analysis progress
4. Confidence scores indicate result reliability
5. All data is fully attributed to sources
